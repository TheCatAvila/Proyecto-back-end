PASOS

1.- Instalar miniconda. (Recordar el PATH)
2.- Instalar django
    pip install django
3.- Crear proyecto
    django-admin startproject NOMBREPROYECTO
4.- Craer APP 
    python manage.py startapp NOMBREAPP (app o core)
5.- para iniciar el proyecto:
    python manage.py runserver
